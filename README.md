# AI Career Mentor

An AI-powered Career Mentor platform to guide students and fresh graduates through career recommendations, skill gap analysis, learning roadmaps, project suggestions, and interview preparation.

## Project Structure

- `backend/` - Spring Boot API with JWT authentication and career mentoring endpoints
- `frontend/` - React app using Material UI, React Router, and Axios

## Getting Started

### Backend
1. Navigate to `backend/`
2. Run `mvn spring-boot:run`

### Frontend
1. Navigate to `frontend/`
2. Run `npm install`
3. Run `npm run dev`

## Features

- User registration and login
- Profile management
- Interest-based career recommendations
- Resume upload and analysis
- Skill gap analysis
- Learning roadmap generation
- Interview preparation guidance
- Job readiness scoring
